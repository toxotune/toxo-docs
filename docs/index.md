Welcome to **TOXO Docs** — the home of the TOXO Python smart‑layer platform.

### What is TOXO?
TOXO lets you train a compact smart layer and attach it to any black‑box LLM (OpenAI, Gemini, Claude). The result is a **CALM (Context Augmented Language Model)**: domain‑aware, preference‑guided, and context‑rich — with zero fine‑tuning.

Formula: **CALM = LLM + TOXO Smart Layer + Context**
